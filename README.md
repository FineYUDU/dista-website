## Translate

1. ```npm install```

2. instal ```npm install typings -g --save-dev``` 

3. ```npm install @types/node --save-dev```

4. in tsconfig.app.json add the next line in "compilerOptions"
```
    "compilerOptions":{
        "types": [ "node" ],
    }
```
